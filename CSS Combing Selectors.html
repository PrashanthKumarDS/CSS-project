Combining Selectors

  1. Element with class selector

    2. p.big{ font-size: 30PX; color: navy;}

    for this need to work Every p that has class="classname"

    This p not have "big" class in it so its not effected

  2. Child selector

    3. article>p{ color: indigo; }

    This line will effected because it is inside the article

    This line will uneffected bcz it is outside the article

    This line will uneffected because it is inside the div of article

  3. Desendant selector

    4. section p{ color: rgb(243, 10, 22); }

    This line will effected because it is inside the section

    This line will uneffected bcz it is outside the section

    This line will effected even it is inside the div of section
